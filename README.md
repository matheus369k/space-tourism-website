## Descrição

A aplicação é um guia para interessados no nosso programa de viagens espacias. Tendo como objetivo sanar as duvidas referentes os destinos, tripulação e naves disponíveis.

Acesse site do guia [space tourism](https://space-tourism-guide.vercel.app/).

## Iniciar

È necessário ter instalado: Nodejs e Git, instalados na maquina.

Clone o repositório localmente.

```bash
git clone https://github.com/matheus369k/space-tourism-website.git
cd ./space-tourism-website
```

Instale as dependências.

```bash
yarn
```

Rode a aplicação

```bash
yarn dev
```
